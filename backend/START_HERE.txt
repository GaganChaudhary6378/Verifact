🎯 WHAT THIS DOES:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Receives claim from frontend (WebSocket)
2. Processes claim (normalize, extract entities, classify)
3. Retrieves evidence from multiple sources (KB + Web)
4. Scores sources by credibility
5. Detects stance (supports/refutes) using NLI
6. Calculates weighted consensus
7. Evaluates quality with RAGAS
8. Returns verdict: TRUE | FALSE | MISLEADING | NOT ENOUGH EVIDENCE
